# RAG Assistant

Локальный интеллектуальный ассистент на основе RAG для поиска и анализа файлов (PDF, TXT, DOCX, PNG, JPG).

## Установка
```bash
poetry install
```

## Запуск
```bash
python src/main.py
```

## Этапы разработки
1. Базовый RAG (индексация, поиск).
2. UI (PyQt5, чат).
3. Интеграция с ОС (ПКМ).
